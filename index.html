Thumb Speed Test

A Micro:bit reaction game — how fast is your thumb?

A simple but fun reaction game for the BBC Micro:bit. Tap Button A as many times as possible in 10 seconds. The LED display lights up when the game starts, counts every tap with a beep, then scrolls your score at the end. Built with MakeCode blocks and TypeScript — includes full source code you can paste straight into the editor.

Thermometer

A servo-powered temperature gauge using the Micro:bit and Sensor:bit board

Build a physical thermometer that moves a servo motor based on the real-time temperature read by the Micro:bit. The servo sweeps from 0° to 180° as the temperature rises from 10°C to 40°C, giving you a live analogue needle readout. Includes full MakeCode blocks and source code.
